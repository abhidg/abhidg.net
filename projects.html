  • Probability distributions of extremal fitnesses in the block model

    project guide Dr. Kavita Jain, JNCASR Bangalore
    The block model is a model to describe fitnesses in biological populations. In this work, we have constructed the extremal (maximum) probability distributions for d mutations. For specific values of the model parameters, scaling limits were also obtained.

  • Machine Recognition of Printed and Handwritten Mathematical Expressions - A Tree Matching Algorithm for Performance Evaluation

    project guide Dr. Utpal Garain, ISI Kolkata
    Mathematical expressions can be represented in terms of MathML files which are essentially trees. In this project, we worked on adapting an existing algorithm for comparing trees to the problem of comparing MathML files. The work has practical applications has applications in doing a quantitative comparison of recognisers for different mathematical expressions. The work was done with Kunal Sain of GCETTB, West Bengal.

    paper K Sain, A Dasgupta and U Garain, Int. Jour. of Document Analysis & Recognition (2010) doi: 10.1007/s10032-010-0121-9
